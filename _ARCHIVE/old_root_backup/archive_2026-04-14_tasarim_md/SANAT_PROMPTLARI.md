## ANİMASYON NOTLARI

### Karakter Animasyon Setleri (her class için)
```
- Idle:        4-6 frame loop (hafif nefes alma)
- Walk:        6-8 frame (yön: 8 yön veya 4 yön)
- Attack:      5-7 frame (hızlı windup + impact + recovery)
- Dash/Dodge:  4-5 frame (afterimage bırakabilir)
- Hurt:        3 frame (kısa flash)
- Death:       8-10 frame (collapse animasyonu)
- Cast/Skill:  skill başına ayrı (4-8 frame)
```

### Efekt Animasyonları
```
- Hit spark:      4 frame (impact noktasında)
- Blood splatter: 4 frame (düşman vurulunca)
- Dash trail:     3 frame afterimage (transparan fade)
- Skill impact:   skill başına (genellikle 6-8 frame radial burst)
- Level up:       8 frame golden sparkle
- Map reveal:     ink spread animasyonu, 10-12 frame
```

### Tip Animasyon Promptu (Stable Diffusion / sprite sheet)
```
prompt: pixel art sprite sheet, [karakter adı] [aksiyon],
8 frames horizontal layout, dark fantasy game art,
32x32 per frame, consistent style,
transparent background, no antialiasing,
clean frame separation
```
