# Image 2 Single Class Production Message
> Use this after the 10-class board direction is accepted. Generate one class at a time, starting with Warblade S.

```text
Use the attached Warblade reference image ONLY as camera / perspective / game-sprite scale reference.
Do NOT copy its character design, weapon, pose, outfit, colors, face, or body rotation.

Use the attached class prompt as the character design instruction.

CAMERA LOCK - FINAL SPRITE BASE:
Generate a single SOUTH/front full-body character concept for a high top-down ARPG game sprite.
The character faces SOUTH/front directly: hips and shoulders square to the camera plane.
The camera is above the character and slightly in front, looking downward like an ARPG dungeon camera.

This must NOT be eye-level portrait art.
This must NOT be a straight front concept-art lineup.
This must NOT be a heroic poster pose.
This must NOT be isometric tile/environment art.

MUST SHOW:
- top/crown of head or hair
- top planes of both shoulders
- top surfaces of boots/feet
- mild face foreshortening
- subtle head-to-feet perspective taper
- full body visible, no cropping

HEAD ORIENTATION:
Do NOT fake the camera by making the character bow.
The head stays upright on the neck.
The face remains partially readable from the front.
Eyes look slightly downward-forward into the battlefield, not at the floor and not direct portrait eye contact.
The camera moves higher; the character does not bend the neck downward.

STYLE / ASSET RULES:
Pure black background only.
No floor, no shadow, no environment, no UI, no text.
Base sprite is not actively casting.
Strong spell colors are reserved for later VFX/cast frames.
Readable 128x128 pixel-art conversion: large shapes, clean silhouette, no micro-detail.

If the result looks like a front-facing character lineup, regenerate with a higher camera.
If the result shows bowed heads instead of a higher camera, regenerate with upright heads.
```
