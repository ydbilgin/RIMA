## Workflow

1. PixelLab UI → Create / Generate (style ref **YOK**)
2. Output: 64×64 · 4×4 (16 cell)
3. Description = yukarıdaki blok kopyala
4. Run → 16 cell sheet döner
5. Sheet → Claude'a at → cell-by-cell QC (kamera + identity + per-class lock + face quality + NO TEXT)
6. PASS cell'ler havuz: kullanıcı PixelLab UI'da o look'a benzeyen tek-class **Create Character standart mode** ile base char üretir → `character_id` alır
7. FAIL cell'ler: per-class single-char re-roll (sadece bozuk olanlar, ucuz)
8. Tüm 10 class + 6 mob/boss `character_id` PASS → Claude MCP `animate_character` ile id üzerinden Run/Attack/Idle/Hit/Death

## QC Eşiği

- 10+/16 PASS → havuz hazır, per-class re-roll minimum
- 6-9/16 PASS → orta, FAIL'ler için targeted re-roll
- 0-5/16 PASS → recipe revize (Description tweak veya style ref ekle)
