## 3. Counter Ayrimi

Verdict: KATILIYORUM, ama uygulama sarti var.

Teorik ayrim dogru:
- Warblade: absorb / break / dominate
- Ronin: wait / draw / punish
- Brawler: weave / evade / body-shot

Bu uclu player'a farkli hissettirebilir, ama sadece tooltip'te ayrilmasi yetmez. Input ve animasyon da farkli olmali.

Warblade counter:
- Hold veya timed guard gibi hissedebilir.
- Vurus gelir, Warblade darbeyi karsilar.
- Feedback: agir metal impact, kisa hit-stop, armor crack, hedefte Broken/Sundered.
- Oyuncu hissi: "Bana vurdun, ben seni kirdim."

Ronin counter:
- Pre-commit / sheathed stance.
- Oyuncu saldiri gelmeden once bekler.
- Perfect draw window vardir.
- Feedback: sessizlik, ince cizgi, sonra tek kesme.
- Oyuncu hissi: "Yanlis anda saldirdin, seni okudum."

Brawler counter:
- Dodge/parry degil, last-second body movement.
- Vurus whiff olur, Brawler aci yakalar.
- Feedback: ayak kaymasi, omuz/dirsek/body-shot.
- Oyuncu hissi: "Saldirini bosalttim, ritmini caldim."

Risk: Uc skill de "0.5s pencere ac, vurulursa counter damage" olarak kodlanirsa uc class ayni olur.

Rule:
- Warblade: impact-based
- Ronin: pre-draw timing
- Brawler: whiff/evade timing

## 4. Warblade Iron Crush

Verdict: KISMEN

Iron Crush'i duz damage buff'tan cikarip Broken hedefte armor shard / catlak yayma yapmak dogru yon. RIMA tonuna daha uygun.

Daha iyi final:

Net karar: Iron Crush redesign dogru, ama "armor shard damage" tek basina yetmez. State window acan skill olmali.

## 5. Gunslinger Execute

Verdict: KATILIYORUM

Deadshot kalmali. Point Blank Execute, Hip Shot'in yakin-mesafe finisher branch'ine kaynamali.

Gerekce:
- Gunslinger identity: slide, shoot, reload, line aim, last bullet / perfect reload.
- Deadshot bu identity'ye daha iyi hizmet ediyor.
- Point Blank Execute ayri skill kalirsa Gunslinger'i melee finisher class gibi gosterir ve Brawler/Warblade/Shadowblade alanina girer.

Hip Shot branch:
- Yakinda kullanilirsa Point Blank finisher olur.
- Perfect Reload veya Last Bullet varsa hedefi geriye iter.
- Brawler Launched / Warblade Broken hedefte ekstra stagger verebilir.
- Ayri skill slotu yemez.

Net:
- Deadshot = uzak, line, precision execute.
- Hip Shot branch = yakin riskli panic/finisher.

## Ekleme Risk

1. State UI overload riski
   - 25 state isim olarak iyi ama oyuncu ayni anda 8 ikon gorurse sistem okunmaz.
   - Public/internal state ayrimi sart.

2. State symmetry trap
   - Her class'a 2-3 state verme zorunlulugu yanlis olabilir.
   - Bazi class'lar 1 cok guclu state ile daha iyi calisir: Shadowblade/Rift Scar, Ronin/Opened, Brawler/Launched.

3. Cross-class matrix scope explosion
   - 10 class icin yonlu cross-class dusunulurse 90 kombinasyon hissi dogabilir.
   - Oneri: full matrix yerine state response families yaz: Broken consumers, Marked consumers, Rift Scar consumers, Corpse Field consumers, Hexed consumers.

4. Animation cost risk
   - State-based sistem guzel ama her state ayri VFX/animation isterse scope patlar.
   - Her state icin minimum readable VFX standardi belirlenmeli.

5. Boss compatibility risk
   - Hard CC state'leri boss'larda birebir calisamaz.
   - Her state contract'ta Boss Rule zorunlu olmali.

6. Demo scope risk
   - Faz 1 demo icin Warblade disinda tum sistemi ayni anda revize etmek riskli.
   - Once core system ve 2-3 class ile test edilmeli.

7. Skill slot reason riski
   - Her skill ozel olmali ama mini-sistem olmamali.
   - Onerilen class ici agirlik: 4 simple reliable, 4 combo/setup, 2 risk/reward, 2 advanced/cross-class.

8. Elementalist shape scope riski
   - Faz 1 icin shape-lite yeter: wall pulse, rune anchor, delayed detonation, simple beam interaction.
   - Gercek reflection/refraction later.

9. Hexer behavior complexity risk
   - Faz 1 effects basit olmali: attack delay, aim wobble, heal punish, dash punish, stack threshold burst.

10. Summoner readability risk
   - Minion, corpse field, sacrifice mark, command anchor, cross-class field ayni anda ekranda olursa okunurluk bozulur.
   - Summoner VFX budget sart: max active visual fields, corpse field duration, minion outline priority, sacrifice pre-flash.
