## Genel Sonuc

Sinif sisteminin ana yonu dogru. RIMA'nin "Fractured Epic" tonu, class verb tablosu ve 4/6 aktif slotlu roguelite build yapisi genel olarak iyi kurulmus.

Ana risk: Bazi skill gruplari hala generic ARPG/MMO kokuyor; bazi siniflarda mekanikler birbirini tekrar ediyor; bazi skill'ler 4/6 slot sisteminde "bunu neden seceyim?" sorusuna yeterince ozel cevap vermiyor.

En buyuk problem: skill sayisi fazla ama davranis farki az olabilir. RIMA'da her skill sadece hasar veya alan efekti vermemeli; class verb'e, fracture temasina, dusman sorularina veya cross-class sinerjiye net hizmet etmeli.

## 1. Warblade

Warblade tona en uygun siniflardan biri. "Engage / break / execute" kimligi net. Rage'in hasar vererek dolmasi Ravager'dan ayrisiyor ve korunmali.

Risk: Kontrol etkileri fazla yigiliyor:
- Iron Charge
- Gravity Cleave
- Earthsplitter
- Iron Counter

Bu skill'lerin hepsi stun / pull / knockup / stagger / punish alaninda calisiyor. Warblade "her seyi CC'leyen melee class" olursa Brawler ve Ronin'in yakin dovus alanini yer.

Oneri:
- Warblade CC krali degil, enemy-state opener olmali.
- Broken Armor, Shattered Armor, Exposed, Staggered gibi cross-class okunabilir state'ler uretmeli.
- Diger class'lar bu state'lere davranissal cevap vermeli.

Problemli skill:
- Iron Crush duz damage buff gibi kaliyor.
- "Tum hasar +%30" RIMA icin fazla generic.
- Kalacaksa sadece damage buff olmamali; kirilmis zirhli hedeflere vurunca catlak yayma, armor shard dusurme veya execute penceresi acma gibi davranis eklenmeli.

Death Blow:
- HP<%30 execute tek basina fazla klasik.
- RIMA'ya daha uygun sart: Broken / Sundered / Staggered / Rift-marked hedefe infaz.

Koru:
- Iron Charge
- Sunder Mark
- Earthsplitter
- Iron Counter fikri
- Death Blow fantasy'si

Redesign:
- Iron Crush
- Death Blow kosulu
- CC yogunlugu

Cross-class adaylari:
- Warblade + Shadowblade: Staggered hedefte Rift Scar collapse.
- Warblade + Brawler: Broken hedef launch edilirse wall-slam bonus.
- Warblade + Hexer: Sundered hedef daha hizli Hex stack alir.
- Warblade + Elementalist: Broken Armor hedefe elemental detonation ekstra fracture damage verir.

Frost tekrar riski:
- Glacial Spike
- Frozen Orb
- Frost Wall
- Blizzard

Radiant tekrar riski:
- Prism Beam
- Solar Flare
- Radiant Pillar

Oneri:
- Elementalist'in "shape" verb'i daha guclu kullanilmali.
- Element var ama sekil manipulasyonu yeterince ozel degil.

Eklenebilecek davranislar:
- Frost Wall'dan Fireball sektirme.
- Frozen Orb uzerinden Blink atinca sadece patlama degil, orb yonu degistirme.
- Meteor'un onceden yerlestirilen rune'a dusmesi.
- Prism Beam'in Frost Wall'dan kirilarak yon degistirmesi.
- Lightbreak sirasinda Fire/Frost spell'lerin tamamen farkli davranmasi.

Problem:
- Lightbreak sistemi fazla muhasebe gibi olabilir: Fire stack, Frost stack, Resonance, Light State, Light stack.

Koru:
- Element Switch
- Lightbreak fikri
- Trinity Storm
- Fire/Frost/Light ucgeni

Redesign:
- Frost skill'lerin davranis ayrimi.
- Radiant skill'lerin davranis ayrimi.
- Element Charge duz buff gibi kalmamali.

Cross-class adaylari:
- Elementalist + Ranger: trap icine element yukleme.
- Elementalist + Summoner: minyonlara element state verme.
- Elementalist + Hexer: Hex stack fazina gore element reaksiyonu degistirme.
- Elementalist + Gunslinger: Burning Ammo sadece fire DoT degil, aktif elemente gore ammo modu olmali.

## 3. Shadowblade

Shadowblade redesign dogru yonde. Eski generic rogue kit'inden cikarak phase / scar / collapse sistemine gecmesi RIMA icin iyi.

Koru:
- Sever kaynaginin pozisyonel olmasi.
- Rift Scar.
- Scar collapse.
- Veil Flicker.
- Seam Rend.
- Night Aperture.

Guclu fikir: Shadowblade sadece hizli dagger class olmamali; mekanda yara birakan, dusmanin icinden gecerek pozisyonu silaha ceviren class olmali.

Tekrar riski:
- Veil Flicker
- Phase Step
- Seam Rend
- Twin Carve

Ayrim onerisi:
- Veil Flicker: kisa phase + Scar uygulama.
- Phase Step: savunma/reposition.
- Seam Rend: dash attack + delayed Scar.
- Twin Carve: combo finisher / taraf degistirme.

Problemli skill:
- Smoke Veil single-player'da net tanimlanmali.
- Enemy aggro/target sistemi yoksa stealth alani neyi kandiriyor: projectile, lock-on, enemy tracking?
- Smoke Veil sadece gorunmezlik alani kalirsa generic rogue skill olur.

Eksik:
- Scar management daha guclu olmali.
- Scar'i erken patlatma.
- Scar uzerinden teleport.
- Scar'i dusmandan zemine tasima.
- Scar sayisi arttikca risk/odul.
- Collapse zinciriyle V meter doldurma.

Cross-class adaylari:
- Shadowblade + Warblade: Staggered hedefte Scar collapse.
- Shadowblade + Hexer: Scar collapse Hex stack'i tek hedefte sikistirip Pandemic ile yayar.
- Shadowblade + Gunslinger: Smoke Grenade / Smoke Veil uzerinden stealth shot.
- Shadowblade + Ronin: Afterimage ve Scar farkli ama birbirini tetikleyen pozisyon araclari olabilir.

## 4. Ranger

Ranger'in mark / trap / detonate yonune cekilmesi dogru. Eski klasik hunter hissinden uzaklasmali.

Koru:
- Focus'un mesafeyle dolmasi.
- Mark sistemi.
- Trap sistemi.
- Detonate fikri.

Problem:
- Mark tarafi fazla yogunlasabilir: Marked Detonate, Predator's Mark, Multi-Mark, Final Strike.
- Bunlar tek sistemin parcalari ama 12 skill icinde fazla yer kaplarsa oyuncu hep "mark paketi" seciyor gibi hisseder.

Movement tekrar riski:
- Tactical Roll
- Hunter's Step
- Rift Step

Ayrim onerisi:
- Tactical Roll: savunma / dodge.
- Hunter's Step: mesafe koruma / Focus kazanimi.
- Rift Step: trap/mark setup reposition.

Eksik:
- Trap tarafi daha guclu olmali.
- Trap verb'i listede var ama skill set icinde mark kadar baskin hissetmeyebilir.

Oneriler:
- Hat/tel tuzagi.
- Dusman gecince mark transfer eden trap.
- Elementalist ile element yuklenebilen trap.
- Hexer ile curse yayan trap.
- Brawler ile launch bonusu veren snare trap.

Problemli execute:
- Final Strike sadece dusuk HP execute olmamali.
- Mark sayisi, trap tetiklenmesi veya uzun mesafe sartiyla calismali.

Cross-class adaylari:
- Ranger + Elementalist: elemental trap.
- Ranger + Gunslinger: mark'li hedefe ricochet / deadshot chain.
- Ranger + Hexer: mark detonation Hex spread yapar.
- Ranger + Brawler: trap'e dusen hedef launch edilirse ekstra wall-slam.

## 5. Ravager

Ravager fantasy'si RIMA'ya uygun: suffer / trade / frenzy. Fury'nin hasar alarak dolmasi Warblade'den net ayriliyor ve korunmali.

Koru:
- Fury = hasar alarak dolma.
- Blood Pact.
- Bloodied Roar.
- Berserk / frenzy fantasy.
- Carnage Spin tarzi vahsi skill'ler.

Ana risk:
- "Hasar al -> guclen" sistemi oyuncuyu kotu oynamaya tesvik edebilir.
- Ravager bilerek dayak yiyen class degil, kontrollu aci ticareti yapan class olmali.

Oneri:
- Perfect hurt window.
- Son anda alinan hasar daha cok Fury verir.
- Self-damage ile Fury, ama sonradan kan borcu odetir.
- Blok/parry degil, vurusu kabul edip karsiliginda daha buyuk siddet uretme.

Tekrar/cakisma riski:
- Warblade Last Stand ile Ravager sustain/fury fantasy'si cakisabilir.
- Ravager cok sustain alirsa Warblade'in dayanikli savasci alanini yer.
- Summoner Dark Pact gibi HP harcama sistemleriyle abuse riski dogar.

Abuse riski:
- Ravager + Summoner: HP kaybi Fury doldurur, sacrifice engine ile sonsuz kaynak / sonsuz burst riski.
- Ravager + Warblade: hem hasar vererek hem hasar alarak kaynak dolarsa "olumsuz blender" riski.

Cross-class adaylari:
- Ravager + Summoner: HP trade ama internal CD sart.
- Ravager + Hexer: dusuk HP'de curse etkisi guclenir.
- Ravager + Brawler: perfect hit / weave timing ile frenzy tetikleme.
- Ravager + Warblade: Broken hedefe self-damage harcayarak brutal execute.

## 6. Ronin

Koru:
- Tension.
- Iaido / Draw fantasy.
- Quickdraw.
- Void Cleave.
- Sakura Veil.
- Counter Draw fikri.

Ana risk:
- Counter sistemi Warblade ve Brawler ile cakismamali.

Net ayrim:
- Warblade counter: darbeyi alir, kirar, domine eder.
- Ronin counter: bekler, dogru anda ceker, cezalandirir.
- Brawler counter: weave/body movement timing ile ritim kurar.

Risk:
- Ronin sadece "parry yapinca hasar vurur" olursa generic samurai olur.
- Ronin'in olayi bekleme gerilimi ve tek dogru an olmali.

Movement tekrar riski:
- Wind Step
- Phantom Step
- Flash Draw
- Iai Pressure baglantilari

Oneri:
- Shadowblade phase hareketiyle karismamali.
- Ronin hareketi "yok olup gecmek" degil, "olculu adim + draw penceresi" olmali.

Eksik:
- Tension sadece bekleyerek dolmamali.
- Dusman saldirisi whiff ederse Tension.
- Son anda saldiri alanindan cikarsan Tension.
- Kilic kinindayken dusman yakindaysa Tension.

Cross-class adaylari:

- Ronin + Summoner: afterimage minyon sayilip sacrifice edilebilir; bu fikir korunmali.
- Ronin + Warblade: cift counter guclu olur ama abuse riski tasir.

## 7. Gunslinger

Gunslinger ton acisindan dikkat gerektiriyor. Kadin gunslinger identity iyi ama modern/twin-stick arcade hissine kacmamali. "Worn leather, rift-marked outlaw, reload rhythm" cizgisinde kalmali.

Koru:
- slide / shoot / reload verb uclusu.
- Heat / Overheat.
- Reload Dance.
- Hip Shot.
- Rift Dash.
- Deadshot.

Tekrar riski:
- Cursor Storm
- Fan the Hammer
- Suppression Fire
- Ricochet
- Burning Ammo

Risk: Bunlar "daha cok mermi / daha cok alan / daha cok seken mermi" hissine dusebilir. Her biri farkli karar uretmeli.

Oneri:
- Perfect reload.
- Son mermi bonusu.
- Bos sarjorle dash bonusu.
- Reload sirasinda yapilan slide ozel atis uretir.
- Overheat sirasinda bazi skill'ler kapanmak yerine davranis degistirir.

Execute ayrimi:
- Deadshot ve Point Blank Execute ayni execute ailesine giriyor.
- Deadshot: uzak mesafe, line aim, yuksek riskli tek hedef infaz.
- Point Blank Execute: yakinda, tehlikeli, pozisyonel finisher.
- Ayrim net degilse biri birlesmeli veya redesign edilmeli.

Cross-class adaylari:
- Gunslinger + Ranger: marked ricochet.
- Gunslinger + Shadowblade: smoke icinden stealth shot.
- Gunslinger + Elementalist: aktif elemente gore ammo modu.
- Gunslinger + Brawler: point blank shot sonrasi launch / knockback.

Tone uyarisi:
- Gunslinger neon arcade, modern shooter, cyberpunk veya cartoony olmamali.
- RIMA icinde rift kazimali silahlar, kirik dunya haydudu, soguk metal, dusuk VFX cizgisinde kalmali.

## 8. Brawler

Brawler mekanik olarak eglenceli ama ton acisindan riskli. "Brawler" adi casual gelebilir. Amber crack gauntlet / ciplak yumrukla kirik dunyaya kafa tutan fiziksel dovuscu olarak verilirse calisir.

Koru:
- weave / combo / launch verb uclusu.
- Charge sistemi.
- Perfect timing.
- Launch / juggle fantasy.
- Amber knuckle/crack identity.

Ana problem:
- Rotation-lock riski yuksek.
- Charge biriktir, bankala, harca, Overdrive'a aktar, weave yap, combo surdur derken sinif yumruk atan dovuscuden combo muhasebecisine donusebilir.

Oneri:
- Charge sistemi sadelesmeli.
- Ya Charge harcanir ve skill guclendirir.
- Ya Charge bankalanir ve V/Overdrive motoru olur.
- Ikisi birden fazla karar gurultusu yaratiyor.

Tekrar riski:
- Aerial Rave
- Seismic Stomp
- Repulse
- Cyclone Drive

Ayrim onerisi:
- Aerial Rave: tek hedef juggle.
- Seismic Stomp: ground crack / anti-swarm.
- Repulse: savunma amacli alan acma.
- Cyclone Drive: combo extender / reposition.

Counter cakismasi:
- Counter Blow, Warblade Iron Counter ve Ronin Counter Draw ile karisabilir.
- Brawler counter vurusu karsilamak degil, weave ile bosa dusurup body-shot punish olmali.

Eksik:
- Dusmani duvara carpma.
- Dusmani baska dusmana firlatma.
- Zemin catlagi ustunde combo uzatma.
- Armor plate sokme.
- Launch edilen dusmana cross-class trigger acma.

Cross-class adaylari:
- Brawler + Warblade: Broken target launch.
- Brawler + Ranger: trap'e dusen hedef juggle.
- Brawler + Ravager: perfect hit sonrasi frenzy.
- Brawler + Summoner: minyonu firlatma / sacrifice projectile gibi ozel ama kontrollu fikirler.

## Genel Cakisma Problemleri

### 1. Counter / Parry Fazlaligi

Warblade, Ronin ve Brawler ayni savunma/punish alanina yaklasiyor.

Net ayrim:
- Warblade: darbeyi karsilar, kirar, dominate eder.
- Ronin: bekler, dogru anda ceker, cezalandirir.
- Brawler: weave ile vurusu bosa dusurur, ritimden punish cikarir.

### 2. Dash / Step / Phase Fazlaligi

Shadowblade, Ranger, Ronin ve Gunslinger hareket skill'leri birbirine yaklasabilir.

Net ayrim:
- Shadowblade: dusmandan gecer, Scar birakir, collapse uretir.
- Ranger: mesafe korur, trap/mark setup kurar.
- Ronin: olculu adim, draw timing, punish penceresi.
- Gunslinger: slide/reload/shoot ritmini degistirir.

Dash + damage olan skill'ler azaltilmali veya class verb'e baglanmali.

### 3. Execute Fazlaligi

HP<%30 execute riski olan siniflar:
- Warblade Death Blow
- Ranger Final Strike
- Gunslinger Deadshot / Point Blank Execute
- Ronin Flash Draw
- Shadowblade Severance

Her execute kendi class sartini istemeli:
- Warblade: Broken / Sundered hedef.
- Ranger: Marked + trap tetiklenmis hedef.
- Gunslinger: son mermi / perfect reload / line aim.
- Ronin: Tension / draw window.
- Shadowblade: Scar collapse sonrasi.

### 4. Cursor AoE / Zone Fazlaligi

Elementalist, Gunslinger, Ranger, Hexer ve Summoner'da cok fazla "cursor alan koy, bekle, patlat" yapisi olusabilir.

Her zone farkli soru sormali:
- Elementalist zone: element sekillendirme.
- Ranger zone: trap / hazirlik.
- Hexer zone: curse pressure.
- Summoner zone: minyon command anchor.
- Gunslinger zone: ates hatti / suppressive control.

### 5. Cross-Class Sistemi

En kritik feedback: Cross-class potansiyeli yuksek ama sadece ornek seviyesinde kalmamali. Her class export edilebilir state uretmeli.

Ornek export state'ler:
- Warblade: Broken, Sundered, Staggered.
- Elementalist: Burning, Frozen, Charged, Lightbroken.
- Shadowblade: Rift Scar, Collapsing, Severed.
- Ranger: Marked, Snared, Exposed.
- Ravager: Bloodied, Frenzied, Wounded.
- Ronin: Opened, Punish Window, Afterimage.
- Gunslinger: Suppressed, Exposed Line, Overheated Ammo.
- Brawler: Launched, Juggled, Wall-Slammed.
- Summoner: Bound, Sacrifice Mark, Corpse Field.
- Hexer: Hexed, Overloaded, Cursed Link.

Cross-class pasifler bu state'lere davranissal cevap vermeli. "+%10 damage" gibi pasifler istenmiyor.

## En Acil Duzeltilecekler

1. Brawler rotation-lock azaltilmali.
2. Hexer rotation dongusu daha davranissal yapilmali.
3. Warblade / Ronin / Brawler counter ayrimi netlestirilmeli.
4. Shadowblade / Ranger / Gunslinger / Ronin movement skill'leri ayrilastirilmali.
5. Execute skill'leri HP<%30 klisesinden cikarilmali.
6. Elementalist frost/radiant skill'leri davranis olarak ayrilastirilmali.
7. Summoner sacrifice engine abuse kontrolu yapilmali.
8. Gunslinger "daha cok mermi" class'ina donusmemeli.
9. Ranger trap tarafi guclendirilmeli.
10. Cross-class sistemi state tabanli yeniden kurulacak kadar netlestirilmeli.
