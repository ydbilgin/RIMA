Codex karari:

- Gerekirse 1 secondary/internal state ekle.
- Full game icin 2-3 state kabul edilebilir, ama public state sayisi cap'lenmeli.

  - Abuse cap

## Additional Systemic Risks

### 1. State Symmetry Trap

Every class does not need equal state count. Equal state count creates artificial design. Some classes should be narrow and sharp:
- Shadowblade can center on Rift Scar.
- Ronin can center on Opened/Draw Window.
- Brawler can center on Launched.

### 2. Public UI Overload

State count must be capped by what the player can read. If 6-8 icons can stack on a target, the system is already failing.

Need a rule:
- max visible target state icons
- priority order
- hidden/internal state policy

### 3. Boss Compatibility

Every state needs Boss Rule. Hard CC cannot map 1:1 to bosses.

Examples:
- Launched -> boss micro-stagger
- Snared/Trapped -> boss dash cooldown delay, not full root
- Suppressed -> projectile spread/recovery modifier
- Hexed -> threshold trigger, no pattern-breaking delay unless boss design approves

### 4. Cross-Class Matrix Explosion

Do not design 90 pair-specific interactions. Use state response families:
- Broken consumers
- Marked consumers
- Rift Scar consumers
- Corpse Field consumers
- Hexed consumers

This keeps cross-class rich without making every pair unique.

### 5. Summoner Economy

Summoner needs its own economy file or section before skill redesign:
- minion cap
- corpse field cap
- sacrifice per second cap
- post-sacrifice recovery
- summon capacity penalty
- cross-class trigger ICD
- boss/elite rules

### 6. Elementalist Scope

Shape interactions are right, but reflection/refraction is later. Faz 1 shape-lite:
- wall pulse
- rune anchor
- delayed detonation
- simple beam-wall burst

### 7. Hexer AI Complexity

Behavioral curse is right, but mind-control or enemy-friendly-fire can be expensive. Faz 1 effects:
- attack delay
- aim wobble
- heal punish
- dash punish
- threshold burst

### 8. Warblade Demo Scope

Warblade must not become design churn. Faz 1 goal:
- keep engage/break/execute feel
- redesign Iron Crush minimally into Sundered window
- gate Death Blow
- prevent CC spam
