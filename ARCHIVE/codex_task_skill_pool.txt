## Design rules for alternatives
- Each alternative MUST offer a DIFFERENT PLAYSTYLE vs the base skill in that slot
  - If base Q is AoE -> alt-1 is single-target burst, alt-2 is a mobility variant
  - If base E is defensive -> alts are offensive or utility variants
- Names follow dark relic aesthetic (same as canonical doc: evocative, slightly archaic)
- Do NOT duplicate any skill name already in the canonical doc
- Must fit class identity (Warblade=melee chain, Specter=drain-based, Riftstalker=portal-anchor, etc.)
- Cross-class synergies optional but encouraged where it makes sense
- F-Tier2 must be a power upgrade/extension of the base ultimate (not a replacement)
