Domain-warped Perlin floor variant painter (Editor-only static class).

Signature:
namespace RIMA.Editor.RoomDesigner
{
    using RIMA.Runtime.Rooms;
    using UnityEngine;
    using UnityEngine.Tilemaps;

    public static class FloorVariantPainter
    {
        private const float AccentThreshold = 0.65f;
        private const float HeroThreshold = 0.88f;
        private const float WarpFreq = 0.05f;
        private const float WarpStrength = 4.0f;
        private const float ZoneFreq = 0.05f;
        private const float DetailFreq = 0.22f;

        public static bool BakeVariants(Tilemap floorTilemap, RoomBlueprint bp, TileBase[] variantSet) { ... }
        public static bool PreviewVariants(Tilemap floorTilemap, RoomBlueprint bp, TileBase[] variantSet) { ... }
        public static void RestoreDefault(Tilemap floorTilemap, TileBase baseTile) { ... }
    }
}

Algorithm (both Bake and Preview):
1. Get all occupied cells from floorTilemap via floorTilemap.cellBounds
2. For each cell (x, y) in bounds:
   a. Check floorTilemap.GetTile(cell) != null — skip empty cells
   b. Compute domain-warp Perlin:
      int seed = bp.noiseSeed;
      float seedOffset = seed * 0.1f;
      float wx = Mathf.PerlinNoise((x + 5.2f + seedOffset) * WarpFreq, (y + 1.3f + seedOffset) * WarpFreq) * WarpStrength;
      float wy = Mathf.PerlinNoise((x + 9.2f + seedOffset) * WarpFreq, (y + 2.8f + seedOffset) * WarpFreq) * WarpStrength;
      float zone = Mathf.PerlinNoise((x + wx) * ZoneFreq, (y + wy) * ZoneFreq);
      float detail = Mathf.PerlinNoise((x + wx + seedOffset) * DetailFreq, (y + wy + seedOffset) * DetailFreq);
   c. Determine tier from zone value: zone < AccentThreshold => base, zone < HeroThreshold => accent, else hero
   d. Cluster-bust: if hero tier AND any neighbor in 8-directions is also hero => downgrade to accent
   e. Pick variant index within tier using deterministic hash:
      base: indices 0-7 (8 tiles), accent: 8-12 (5 tiles), hero: 13-15 (3 tiles)
      uint hash = (uint)((x * 73856093) ^ (y * 19349663) ^ seed);
      int varIdx = tierOffset + (int)(hash % (uint)tierSize);
   f. Place: floorTilemap.SetTile(cell, variantSet[varIdx])
   g. For Bake only: write to bp.floorVariantIndex (ensure array is sized roomWidth * roomHeight, store at index = (y - bp.roomOrigin.y) * bp.roomWidth + (x - bp.roomOrigin.x))
3. floorTilemap.RefreshAllTiles()

Cluster-bust: implement private static bool HasHeroNeighbor(Tilemap tilemap, Vector3Int cell, RoomBlueprint bp, int seed) — checks 8 neighbors, computes zone using same Perlin formula, returns true if any neighbor zone >= HeroThreshold.

BakeVariants: if variantSet is null or length == 0, return false immediately. Otherwise run algorithm and return true.
PreviewVariants: same as BakeVariants but skip writing bp.floorVariantIndex. Return true on success.
RestoreDefault: iterate all cells in floorTilemap.cellBounds, if tile != null set to baseTile, then RefreshAllTiles.

namespace RIMA.Editor.RoomDesigner
{
    using System.Collections.Generic;
    using RIMA.Runtime.Rooms;
    using UnityEngine;
    using UnityEngine.Tilemaps;

    public static class WallAutoConnect
    {
        // Connection type indices (match 8-variant tileset order)
        // 0=straight_H, 1=straight_V, 2=corner_NW, 3=corner_NE, 4=corner_SW, 5=corner_SE, 6=end_L, 7=end_R

        private static readonly Vector3Int[] NeighborOffsets = {
            new Vector3Int(-1, 0, 0),   // W
            new Vector3Int(1, 0, 0),    // E
            new Vector3Int(0, 1, 0),    // N
            new Vector3Int(0, -1, 0),   // S
            new Vector3Int(-1, 1, 0),   // NW
            new Vector3Int(1, 1, 0),    // NE
            new Vector3Int(-1, -1, 0),  // SW
            new Vector3Int(1, -1, 0),   // SE
        };

        public static void RefreshNeighborhood(Tilemap wallTilemap, IEnumerable<Vector3Int> touchedCells, TileBase[] wallVariants, RoomBlueprint bp = null) { ... }
        public static void RefreshCell(Tilemap wallTilemap, Vector3Int cell, TileBase[] wallVariants, RoomBlueprint bp = null) { ... }
        public static bool BakeWallVariants(Tilemap wallTilemap, RoomBlueprint bp, TileBase[] wallVariants) { ... }
    }
}

Connection logic for RefreshCell:
1. Check if cell has a wall tile (GetTile != null) — if not, skip
2. Check override: if bp != null, compute cellIdx = (cell.y - bp.roomOrigin.y) * bp.roomWidth + (cell.x - bp.roomOrigin.x); if cellIdx is in range and bp.overrideVariantIndex[cellIdx] => skip
3. Build neighbor mask (4-directional NSEW only):
   bit 0 = W has wall, bit 1 = E, bit 2 = N, bit 3 = S
4. Determine connection type from 4-bit mask:
   0b0000 (0)  = isolated        => idx 0
   0b0011 (3)  = E+S             => idx 5 (corner_SE)
   0b0101 (5)  = N+S             => idx 1 (straight_V)
   0b0110 (6)  = E+N             => idx 3 (corner_NE)
   0b1001 (9)  = W+S             => idx 4 (corner_SW)
   0b1010 (10) = W+E             => idx 0 (straight_H)
   0b1100 (12) = W+N             => idx 2 (corner_NW)
   0b0001 (1)  = S only          => idx 7 (end_R)
   0b0010 (2)  = E only          => idx 6 (end_L)
   0b0100 (4)  = N only          => idx 6 (end_L)
   0b1000 (8)  = W only          => idx 7 (end_R)
   all three-way or four-way masks => idx 0 (straight fallback)
5. If wallVariants is null or variantIdx out of range => find first non-null tile in wallVariants (or null)
6. wallTilemap.SetTile(cell, selectedVariant)

RefreshNeighborhood:
1. Collect all cells in 3x3 region around each touchedCell into HashSet (deduplicate)
2. Call RefreshCell for each collected cell

BakeWallVariants:
- If wallTilemap or bp or wallVariants is null => return false
- Iterate all cells in wallTilemap.cellBounds
- For each cell with a tile, determine connection type using same 4-bit mask logic as RefreshCell
- Write variant index (the int connection type 0-7) as byte to bp.wallVariantIndex (ensure array sized roomWidth * roomHeight)
- Also set tile on tilemap
- Call wallTilemap.RefreshAllTiles()
- Return true
