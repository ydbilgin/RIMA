PROPOSAL UNDER REVIEW
================================================================

CORE PROBLEM: Chibi 64x64 sprite signals "cute/playful" in Western gaming convention (Hammerwatch, Vampire Survivors, Brotato). "Fractured Epic" requires DRAMATIC weight without sliding into grimdark or cute-only. How does the dramatic narrative (Rift March, Architect, 3 endings, 4 multi-phase bosses) integrate with chibi visual identity?

PROPOSAL (7 design decisions):

DECISION 1 — Tonal Model: "HLD-leaning dread", NOT Hammerwatch ironic contrast.
Rationale: Chibi vulnerability = dramatic amplifier (not fought against). Drama lives in env scale (towering walls 2-3 tile tall, decay decals) + boss-vs-player scale (256 sprite vs 64 sprite). The vulnerability IS the drama. Player reads "small fragile thing in vast hostile place".

DECISION 2 — Hybrid Two-Tier Cinematic System:
- Gameplay = chibi 64x64
- Boss intros + 3 ending cinematics + Architect reveal = pre-rendered PORTRAIT/ILLUSTRATION frames (static, "tarot-card" style 256x384 or 512x768)
- Portraits = PixelLab Pro base + manual paint-over. NOT 3D. NOT video.
- Cinematic Layer D upgrades from "frame" → "portrait reveal moment with parallax + slow reveal animation"

DECISION 3 — UI/Tone Language:
- Typeface: serif for codex/lore (gravitas), clean sans for HUD (clarity)
- Prose: terse, archaic, fragment-based (HLD codex style)
- Color grading: gameplay = neon-on-mat (locked palette), UI overlay = muted parchment/glass + neon accent highlights
- Audio flag: low drone + sparse dialogue (dread amplifier)

DECISION 4 — Environmental Storytelling at 32x32:
- Decal pass MANDATORY (blood, claw marks, broken statues, void cracks, ash drift)
- Tile = base ruin, decal = narrative payload
- 30-35° view favors floor decal readability
- Lighting: rim light + flickering point + low-fog ambient
- Tile alone insufficient; decal+light combo carries env story

DECISION 5 — NPC + Hub Portrait System:
- Heavy NPCs (Architect Echo, Penitent Whisper, ending NPCs) = fullscreen portrait modal in plot-heavy dialogue
- Daily hub chitchat = chibi sprite + bubble text
- Portraits = ~512x768 illustration, 8-12 portraits v1 budget

DECISION 6 — Architect Final Reveal:
- 256x256 sprite (4x player chibi-scale)
- Multi-layer compositing: 3 parallax backdrop layers (void/rift/silhouette) + screen-edge crack VFX + letterbox during phase transitions + portrait cut-in on phase change (full-screen illustration 2-3 sec)
- Boss "feels" larger via composition, not sprite size scaling

DECISION 7 — Risk Mitigations:
- "Cute" read risk → env palette muted (locked), char silhouette readable, walk anim grounded not bouncy
- Hub "tavern feel" risk → hub env decay/cracked stone, cold lighting, NPCs scattered not grouped
- Ending payoff weak on chibi avatar risk → ending = portrait-only cinematic, chibi not shown in final beat
- Boss intro gravitas loss risk → portrait system (Dec 2) carries it
