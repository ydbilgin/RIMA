# RIMA UI/HUD Implementation Pack

İçerik:

- `00_current_screenshots/`: Senin attığın mevcut ekranlar.
- `01_improved_mockups/`: Üretilen yeni HUD / ödül / pause / skill codex mockupları.
- `02_asset_specs/`: Hangi UI assetlerinin gerektiği, CSV manifest, renk tokenları, atlas blockout planı.
- `03_unity_notes/`: Unity tarafında nasıl prefab/slice/Canvas kurulacağı.
- `04_reference_sheets/`: Varsa mevcut skill/char/map referans sheetleri.

En önemli dosya: `02_asset_specs/ASSET_CHECKLIST_TR.md`

Kısacası gereken şey “tek tek devasa ekran görseli çizmek” değil. Gereken şey:
1. Ortak 9-slice UI atlası.
2. Hotbar slot seti.
3. Reward card prefab parçaları.
4. Pause modal parçaları.
5. Skill Codex row/tab/scroll parçaları.
6. Minimap frame + markerlar.
7. Debug kırmızı kareleri yok edecek runtime ayarı veya replacement rune sprite.
