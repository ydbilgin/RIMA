## Workflow per prompt

1. Open https://pixellab.ai/app/create
2. Select "Create Object" tool
3. Paste the prompt (Universal prefix + specific prompt)
4. Set canvas (96×96 or 96×128 per prompt)
5. Upload `wall_n_v1.png` as Style reference image (or ChatGPT ref image for stronger Hades-iso anchor)
6. Set View to "High top-down" (~30-35° tilt)
7. Set Background to "Transparent"
8. Generate (n_frames=4 to get 4 candidates → pick best)
9. Download best candidate
10. Rename to target filename (e.g. `corner_SE_v1.png`)
11. Save to `Assets/Art/Walls/Act1_ShatteredKeep/`

After all prompts done → notify orchestrator → Unity import + prefab + WallPrefabRegistry will be dispatched (Codex task).
