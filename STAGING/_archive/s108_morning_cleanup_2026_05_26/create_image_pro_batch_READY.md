## Usage workflow

For each item below:
1. **Reference image:** Attach `Characters/anchors/warblade.png` (or class-specific anchor if listed)
2. **Description field — yapıştır şu sırada:**
   - Senin matching instruction'ın (örn: `"Match exact angle, facing direction, and chibi proportions from the reference image only; do not copy outfit, armor, weapon, build, or pose."`)
   - Aşağıdaki POSE BODY metnini yapıştır
3. **Canvas:** Belirtilen boyutu Create Image Pro'da set et
4. Generate → 16 variations çıkar → en iyiyi seç

**16 generation × ~6 credit = ~96 credit total** (S74 estimate 162'di çünkü 11 weapon dahildi; sadece karakter+mob = 96)

## Matching instruction template (senin yazacağın bölüm)

Yapıştırmadan önce description'ın başına eklemek için 3 alternatif:

**A) Sıkı (recommended):**
```
Match exact angle (35-degree high top-down), facing direction (south), and chibi 64x64 proportions from the reference image only; do not copy the outfit, armor, weapon, build, or pose of the character in the reference.
```

**B) Orta:**
```
Match exact angle and facing direction from the reference image only.
```

**C) Esnek:**
```
Use the reference image only as an angle and proportion guide; vary all other identity features freely.
```

Önerim: **A**. K4 prefix S74 LOCK'taki form bu — Opus judgment ile pasted from `character_idle_LOCK_S74.md`.
