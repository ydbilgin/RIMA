# Unity Tile Palette Workflow - RIMA Combat v15g

Use Blueprint Painter for STRUCTURE and Unity Tile Palette for POLISH.

## Open
- Open `Window > 2D > Tile Palette`.
- Or use `Tools > RIMA > Map Designer > Open Tile Palette`.

## Select Palette
- Use the dropdown at the top of the Tile Palette window.
- Select `RIMA_Combat_v15g`.

## Paint
- Select the brush tool.
- Click a tile in the palette to switch the active tile.
- Click or drag on the target Tilemap for immediate visual feedback.

## Tools
- Erase: select the eraser toolbar button and drag on the Tilemap.
- Bucket fill: select the fill toolbar button and click a connected area.
- Rectangle: select the rectangle toolbar button and drag an area.
- Line: use the line toolbar button when available in the Tile Palette toolbar.

## Categories
- Floor: top row, cobble dominant variants.
- Path: second row, cobble secondary/path variants.
- Dirt: third row, dirt variants.
- Transition: fourth row, cobble transition candidates.
