# Create Character V3 — 10 Class Prompts (Create from Text)
**Karar refs:** #71 positive-only / #80 Class Silhouette Bible / #99 no hand mention / #100 chibi 64x64 / #123 weapon decouple
**Tool:** PixelLab WebUI → Create Character → Create from Text → V3 mode
**Body type:** humanoid
**Size:** 64 pixels (canvas auto-expands to ~90px for animation padding)
**Recommended params:**
- Detail: **medium detail**
- Outline: **single color black outline**
- View: **high top-down** (Hades 35° match)

**Workflow:** Her prompt'u tek tek WebUI'a yapıştır, V3 generate et. Çıktı: south-facing sprite + 8 direction auto-rotation. ~20-40 generation cost per character (V3 Pro), 3-5 dakika async.

**Estimated total:** 10 × ~30 gen = ~300 generation credits, ~30-50 dk sequential, OR paralel sekme'lerle ~10 dk.

## Workflow (her karakter için)

1. WebUI → Create Character → Create from Text → V3 mode seç
2. Body type: humanoid
3. Size: 64 pixels
4. Detail: medium detail
5. Outline: single color black outline
6. View: high top-down
7. Yukarıdaki prompt'lardan birini yapıştır (her birini ayrı session)
8. Generate → 3-5 dk async, 8-direction sprite sheet çıkar
9. Bittikten sonra `mcp__pixellab__get_character` ile çek (PNG + metadata)
10. Beğenmediğin direction varsa regen, ya da tüm character'ı düşük ai_freedom ile yeniden

## V3 vs Create Image Pro v5 — hangisi?

| Yön | Create Image Pro v5 (single batch) | Create Character V3 (per class) |
|---|---|---|
| Direction sayısı | 1 (south-facing single frame) | 8 (south + 7 others auto-rotation) |
| Batch size | 16 sprite tek seferde | 1 character per gen |
| Gen cost | ~6 credit total | ~20-40 credit × 10 = ~200-400 credit |
| Süre | ~30 sn 1 gen | ~30-50 dk 10 gen sequential |
| Karakter identity | Birden fazla'yı tek batch içinde | Her karakter izole, fokus tek |
| Animation hazırlığı | YOK (sadece base) | ✓ 8-direction zaten var, animate_character'a direkt geçer |
| Master sheet kalitesi | ✓ Hızlı keşif | ✓✓ Production-grade |

**Önerim:**
- **Önce v5 single batch (Create Image Pro)** — 16 sprite hızlı keşif, hangi class identities iyi çıkıyor gör
- **Sonra V3 (Create Character) sadece beğenilen 10 karakterin final üretimi** — production 8-direction sheet

Veya direkt V3'e geç, master sheet bypass et. İkisi de geçerli.
