# RIMA - The Shattered Keep / Large Room Layout Pack

Bu paket, Claude / Codex / Unity tarafında doğrudan kullanılabilecek büyük ASCII oda verileri içerir.

## Kurallar

- `.` = walkable floor
- `P` = player entry spawn, exactly 1
- `e` = enemy spawn
- `C` = chest spot
- `B` = boss spawn
- space = void
- Doors only face `N`, `E`, `W`. Never `S`.
- Door coordinates point to an edge floor tile. Door glyphs are not written into the grid.
- Room floors are one connected walkable region; donut rooms intentionally include a center void hole.
