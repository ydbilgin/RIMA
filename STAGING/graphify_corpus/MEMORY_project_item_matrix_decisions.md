# LEGENDARY BALANCE (FINAL)
* Ruin Eternal (Ravager): Kill-gate ext cap +5s

# SYSTEMS & RULES
* Arcane Singularity: 4 Primary skills same element = +40% dmg (Secondary Z/X ignored)
* Elements: Elementalist ONLY. Others use physical/mechanical themes.
* Cross-Class Anim: Body-based adaptation (Option B). Use existing dash/lunge poses. No new sprites.
* Projectiles: PixelLab sprites (4f, 32px) + Unity physics/VFX.
