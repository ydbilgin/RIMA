* **Input:**
  - **LMB:** 3-hit Combo.
  - **RMB:** Secondary identity.
  - **Dash:** Baked-in Dash Attack (No separate slot).
  - **Skills:** 5 slots (Select from 12 per class).
  - **V Burst:** Automatic (Normal) / Interactive (Boss/Elite).
* **V Meter Fill:**
  - Warblade: CC/Parry. Elem: Reactions. Shadow: Scar collapse. Ronin: Deflect.
* **Dash Attacks:**
  - Warblade: Momentum Slam (Dizzy). Elem: Elemental Pulse (AoE). Shadow: Seam Rend. Ronin: Kage Slash.
* **Item System:** Relics (2-3) + Skill Modifiers (2-3). No equipment slots.
