---
name: Prompt — Positive RIMA Spec Mandatory
description: Negations alone lead to generic fallbacks. Positive specs required.
type: feedback
---
* **Rule:** Negations (No Tolkien, etc.) are insufficient. Use 1:3 ratio (1 Negation : 3 Positive).
* **Mandatory Positive Specs:**
  - **Narrative Anchor:** Specific "Fracturing" scar/mark.
  - **Color Placement:** Specific location (e.g., belt stitch, plate crack).
  - **Silhouette Signature:** One unique readable feature (e.g., Warblade ember inner tabard).
  - **Physical Marks:** Detailed description (e.g., "right cheek rift-burn with blue discoloration").
* **Test:** If prompt is generic without negations, positive specs are too weak.
