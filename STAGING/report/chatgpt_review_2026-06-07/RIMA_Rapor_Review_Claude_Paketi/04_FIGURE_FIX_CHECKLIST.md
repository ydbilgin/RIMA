# Şekil Düzeltme Checklist

## Şekil 1 — Attunement Chamber genel görünümü
- [ ] Dummy HP görünmüyor.
- [ ] Debug marker yok.
- [ ] Pedestal/sınıf silüeti gerçekten okunuyor.
- [ ] Caption görseldeki şeyi anlatıyor.

## Şekil 2 — Bürünme anı
- [ ] `[G] Bürün — SINIF` prompt’u okunuyor.
- [ ] Cyan VFX görünüyor.
- [ ] Yazı objelerin arkasında kalmıyor.
- [ ] Debug marker yok.

## Şekil 3 — Combat
- [ ] Aktif combat anı.
- [ ] Death/retry UI yok.
- [ ] Draft UI yok.
- [ ] En az bir düşman hit/knockback/telegraph durumunda.
- [ ] Mümkünse `[RMB] İnfaz` prompt’u görünüyor.
- [ ] Debug marker yok.

## Şekil 4 — Draft
- [ ] 3 kart görünüyor.
- [ ] Tooltip açık.
- [ ] Sinerji chip/satırı okunuyor.
- [ ] Rarity/ikon varsa görünüyor.
- [ ] Debug marker yok.

## Şekil 5 — Portal seçimi
- [ ] UI butonu değil, world-space Rift portalı.

- [ ] Oda türü rünleri görünür.
- [ ] Combat/Elite/Reward/Boss dışı graph’ta olmayan tür yok.
- [ ] Debug marker yok.

- [ ] Görsel placeholder değil.
- [ ] Warblade silüeti okunuyor.
