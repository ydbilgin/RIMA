# RIMA Rapor Review Paketi

Bu zip, Claude'a verilecek acımasız ikinci tur rapor review paketidir.

## Dosyalar

- `01_ACIMASIZ_REVIEW.md`: detaylı cerrahi eleştiri listesi.
- `02_ONCELIKLI_DUZELTME_LISTESI.md`: P0/P1/P2 öncelik sırası.
- `03_CLAUDE_PATCH_PROMPT.md`: Claude'a doğrudan verilecek uygulama prompt'u.
- `04_FIGURE_FIX_CHECKLIST.md`: şekiller için kontrol listesi.
- `manifest.json`: paket özeti.

## Kullanım

Claude'a şu dosyalarla birlikte ver:
1. `RAPOR_RIMA_2026-06-06.docx`
2. `GAME_DESIGN_INTENT_2026-06-06.md`
3. `VISION_VS_CURRENT_2026-06-06.md`
4. Bu zip içindeki markdown dosyaları

Ana hedef: raporu baştan yazmak değil, final teslim öncesi cerrahi düzeltme yapmak.
