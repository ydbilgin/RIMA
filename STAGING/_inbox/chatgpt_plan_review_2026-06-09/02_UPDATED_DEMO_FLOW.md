# Updated Demo Flow

Final akış:

```txt
MainMenu
→ Chamber
→ Combat 1: basit onboarding
→ Combat 2: ikinci wave / biraz baskı
→ Shop: nefes ve seçim
→ Combat 3: shop kararını test ettir
→ Boss: mini boss + victory
→ Victory / Death
```

5 oda demo için doğru uzunluk.

Combat 1: 2-3 kolay mob.
Combat 2: 3-4 mob, 1 ranged veya daha yavaş ama güçlü mob.
Combat 3: 4-5 mob, shop sonrası güçlenmeyi hissettir.
Boss: 1 boss, 2-3 saldırı, HP bar, ölüm → Victory.
