# Shop Minimum Spec

Demo shop = 3 stand.

1. Heal
- Fiyat: 20 Echo
- Etki: Max HP'nin %30'unu iyileştirir

2. Damage Upgrade
- Fiyat: 35 Echo
- Etki: +%12 damage, run sonuna kadar

3. Cooldown Upgrade
- Fiyat: 35 Echo
- Etki: skill cooldown -%10

Echo ekonomisi:
```txt
Combat clear reward: 20 Echo
Shop'a girerken oyuncu: 40 Echo
Heal 20
Damage 35
Cooldown 35
```

Oyuncu ya tek güçlü upgrade alır ya heal alır ya saklar.

Shop'u inventory, relic, item matrix ve meta progression'a bağlama. Demo shop sadece run içi buff versin.
