# Boss Minimum Spec

Mevcut mob'u 1.5-2× büyütmek demo için yeterli, ama sadece scale büyütmek yeterli değil.

Şart:
1. HP bar
2. Telegraph
3. Victory trigger

3 attack:
- Slam Circle: 0.8s telegraph, yakın alan hasarı.
- Line Charge: çizgi telegraph, 0.9s sonra charge, kısa stun.
- Rift Burst: 3 küçük alan sırayla patlar.

Demo için gerçek faz şart değil. HP < 50% cooldownlar %15 hızlanabilir.

Boss ölünce:
- kısa input lock
- VICTORY panel
- Return to Main Menu
- Time.timeScale restore
