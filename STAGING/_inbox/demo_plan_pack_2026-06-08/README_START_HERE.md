# RIMA Demo Plan Pack — 2026-06-08

Bu paket, RIMA için daraltılmış ve gerçekten bitirilebilir bir demo akışı planıdır.

## Demo hedefi

**Amaç:** 2 sınıflı kısa oynanabilir demo.

- Sınıflar: **Warblade + Elementalist**
- Oda akışı: **3-4 combat oda + 1 shop oda + 1 boss oda**
- Sanat/polish: şimdilik minimum. “Make-up pass” sonra.
- Ana odak: akışın çalışması, savaşın okunması, sınıf seçimi, silah bağlama, reward/shop/boss loop.
