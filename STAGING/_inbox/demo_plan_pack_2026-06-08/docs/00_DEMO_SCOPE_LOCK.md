# 00 — Demo Scope Lock

## Kapsam

Demo sadece şu akışı kanıtlar:

```text
Main Menu
→ Character Select / Chamber
→ Warblade veya Elementalist seç
→ Arena run başlat
→ 3-4 combat room
→ 1 shop room
→ 1 boss room
→ boss ölür
→ demo clear screen
```

## Demo sınıfları

### 1. Warblade

Rol:
- Yakın dövüş
- Basit combo
- Ağır ama input gecikmesi olmayan hit hissi
- Greatsword her zaman elde/hazır görünmeli

Minimum skill:
- Basic slash / cleave
- Forward charge veya kısa cone attack
- Sunder mark gibi tek status
- Bir ultimate ya da büyük cooldown skill şart değil, demo için opsiyonel

### 2. Elementalist

Rol:
- Orta menzil projectile/cast
- Alan kontrolü
- Warblade’den net farklı his

Minimum skill:
- Basic fire/arcane projectile
- Small AoE burst
- Slow/freeze veya burn gibi tek status
- Floating rune disc görsel kimlik

## Demo oda sayısı
