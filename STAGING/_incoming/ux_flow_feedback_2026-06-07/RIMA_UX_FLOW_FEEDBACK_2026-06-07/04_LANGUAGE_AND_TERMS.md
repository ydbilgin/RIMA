# 04 — Dil ve Terim Tutarlılığı

## Ana sorun

Ekranlarda TR/EN karışımı çok fazla:

- `BAŞLA / AYARLAR / ÇIKIŞ`
- `SETTINGS`
- `GAMEPLAY / ACCESSIBILITY / AUDIO / CONTROLS`
- `RUN PATH`
- `Skill Codex`
- `MAIN MENU / PLAY AGAIN`
- `TEKRAR DENE / ANA MENÜ`
- `Rift Gir`
- `ODA 1 — ÖDÜL SEÇ`

Bu karışım oyunu erken prototype gibi gösteriyor. Dil karışımı bazen stil olabilir, burada daha çok "henüz toparlanmadı" gibi duruyor. Üzgünüm, menü de insan gibi tutarsız davranınca oyunun havası gidiyor.

## Önerilen karar

Demo Türkçe sunulacaksa bütün UI Türkçe olsun. Steam CTA gibi dış ürün linkleri İngilizce kalabilir.

## Terim standardı

| Şu an | Önerilen TR | Not |
|---|---|---|
| Main Menu | Ana Menü | Her yerde TR |
| Play Again | Tekrar Oyna | Death/Victory |
| Settings | Ayarlar | Başlık dahil |
| Run Path | Koşu Yolu | Map overlay |
| Skill Codex | Yetenek Kodeksi | Ya da demo'da açma |
| Wishlist on Steam | Wishlist on Steam | CTA olduğu için EN kalabilir |
| Rift Gir | Rift'e Gir | Dil hatası |
| Attuned Echo | Echo Büründü | veya `Warblade Echo'suna Büründün` |
| Execute | İnfaz | `[RMB] İnfaz` iyi |
| Reward | Ödül | Chest/Reward için |
| Portal Kapıları | Rift Portalları | Kapı değil portal |
| Chest | Sandık / Ödül | Teknikte Chest, oyuncuya Ödül |
| Boss | Boss | Türkçe oyunlarda kabul edilebilir; alternatif `Patron` ama garip durabilir |

## Kapı vs portal

Artık oyuncuya:
- `Kapı` deme.
- `Rift Portalı` veya sadece `Portal` de.

Metin örnekleri:
- `Bir portal seç`
- `Elite Portalı`
- `Ödül Portalı`
- `Boss Rift'i`
- `Rift'e Gir [G]`

## Kısa transition metinleri

- Chamber giriş: `Bir Echo seç`
- Yaklaşınca: `[E] Bürün: Warblade`
- Bürününce: `Warblade Echo'suna büründün`
- Rift kapısı: `[G] Rift'e Gir`
- Oda başlangıcı: `Oda 1 — Düşmanları Temizle`
- Oda bitti: `Oda Temizlendi`
- Reward: `[G] Ödülü Al`
- Draft: `Bir yetenek seç`
- Portal: `Bir sonraki odayı seç`
- Boss: `Penitent Sovereign yaklaşıyor`
- Victory: `Demo Tamamlandı`
- Death: `Koşu Bitti`

## UI ton kuralı

- Büyük başlıklar: kısa ve net.
- Alt açıklamalar: 1 satır.
- Tutorial prompt: aynı format:
  - `[E] Bürün`
  - `[G] Etkileş`
  - `[RMB] İnfaz`
  - `[M] Harita`
  - `[TAB] Karakter`
- Tuş formatı hep köşeli parantez.
