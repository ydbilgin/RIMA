### Schema v2 (LLM-promptable, Opus şekli + Flash düzeltmeleri)
```json
{ "version": 2,
  "id": "combat_large_donut_01", "roomType": "Combat", "size": {"w": 28, "h": 20},
  "walkable": ["..####....", "  (satır=grid satırı, '#'=walkable, '.'=void; ÜST satır = kuzey)  "],
  "spawn": {"x": 14, "y": 2},
  "exitSlots": { "NW": {"x": 8, "y": 18}, "N": {"x": 14, "y": 19}, "NE": {"x": 20, "y": 18} },
  "props": [ {"id": "<propDefGuid|propName>", "x": 4, "y": 6, "variant": 0, "flipX": false} ],
  "enemySpawns": [ {"x": 8, "y": 14, "tier": "standard"} ],
  "notes": "LLM rationale / designer notu" }
```
Kurallar: bilinmeyen key'ler yok sayılır (forward-compat) · eksik key=default · **Y-FLIP:** JSON satır-0=üst, grid Y=alt-orijin → export `jsonY = h-1-gridY` (Flash bomba #2) · marker karakterleri (P/e/B/C) v2'de AYRI alanlara taşındı (walkable saf '#/.') — eski v1 importer formatı okumaya devam eder · pozisyonel tuple YASAK (LLM çevirir), named key şart.
