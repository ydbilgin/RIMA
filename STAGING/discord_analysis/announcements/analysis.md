### [2026-04-25] Version 0.4.101 & 8-Direction Rotation Update
- Participants: Kaninen
- Type: text + image
- Summary: Kaninen announced Version 0.4.101. Key additions include "Create image S-XL" (32x32 to 512x512) and a new "Rotate to 8 directions" tool that better retains items in the right hand and keeps the pose (supports up to 256x256). An "Object creator" was also released, allowing animation in the online editor similar to the character creator. The Map editor now supports sidescroller perspectives with backgrounds.

### [2026-03-16] Version 0.4.92 & Animate with Text (new)
- Participants: Kaninen
- Type: text
- Summary: Announced Version 0.4.92. A new "Animate with text (new)" tool was added to Aseprite, Pixelorama, and Character Creator. It supports 4-16 frames for sizes up to 256x256 (costs 1-9 generations). Pro tool pricing was updated (API pricing lowered, 20 generations for <=256x256, 25 for <=341x341). ### [2025-11-23] Version 0.4.69 & Original "Animate with Text" Release
- Participants: Kaninen
