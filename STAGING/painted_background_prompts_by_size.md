# Usage Notes

## Workflow
1. Open PixelLab Create Image Pro V3 (web UI)
2. Set size to the value listed under each room heading
3. Leave Style image and Reference image EMPTY
4. Paste the Description block into the main prompt field
5. Paste the Negative Prompt block into the Negative Prompt field
6. Generate -- no AI Enhance on these background prompts

## Size reference (locked valid sizes only)
- 512x288 (16:9) -- standard room width
- 688x384 (16:9) -- large room width
- 512x512 (1:1) -- square chamber (shrine / chest)

## Biome consistency rules
- All prompts use Shattered Ruins / Shattered Keep palette: cool gray granite + cyan rift + moss + rubble
- Variation is achieved through: composition focus, prop silhouettes, light source type and color temperature, rift intensity
- Do NOT mix warm brown palette into ruins rooms -- Merchant_01 uses warm amber lantern as accent only, not floor tone
- Cyan rift intensity scale: Spawn (trace) < Combat_Small (hairline) < Combat_Medium (crack) < Combat_Large (fracture) < Boss (massive fracture)
