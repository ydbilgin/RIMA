## Kimlik
- **Kaynak:** Hex Stacks (0-10) düşman üzerinde
- **Faz sistemi:** 0-3 Debuff / 4-6 Pressure / 7-9 Overload / 10 Hexblast
- **Fantezi:** "Curse yığ → Yay → 10 stack patlat"
- **Silah:** Grimoire / Cursed Totem / Scepter

## 12 Common Liste

### Active (11)
1. **Corruption ★** — anında 3 stack + 4s güçlü DoT
2. **Agony** — durdurulamaz DoT, Pressure → tick artar
3. **Pandemic** — hedefteki TÜM Hex stack'leri etrafa kopyala
4. **Hexblast** — stack durumuna göre patlat, 10 stack → devasa + CD sıfır
5. **Spitback** — kendine gelen saldırının bir kısmı düşmana lanet
6. **Haunt** — hayalet bağla, takip + 3 stack
7. **Unstable Affliction** — heal/dispel → devasa patlama + stun
8. **Enervate** — 10s -%40/%50 hız/saldırı hızı
9. **Foul Wave** — görüntüdeki herkese 2 stack + %8 HP kes
10. **Hex Overload** — Pressure (4+) → hasar başına ekstra stack
11. **Blight Sigil** — zemin curse zone, basana stack birik

### Passive/Echo (S44) (1)
12. **Whisper Mark** — Hexlenmiş yanına gelen otomatik Hex (auto-spread pasif)

## Faz 1 Demo Havuz (5)
1. Corruption (signature stack + DoT)
2. Pandemic (spread mekanik)
3. Hexblast (10 stack payoff)
4. Enervate (control debuff)
5. Blight Sigil (zone curse)
