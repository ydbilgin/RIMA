## Kimlik
- **Kaynak:** Element charge ritmi (Fire ↔ Frost ↔ Lightbreak)
- **Fantezi:** "Fire koy → Frost ile patlat → Lightbreak ortaya çıkar"

## 12 Common Liste

### Active (10)
1. **Fireball ★** — DoT + Fire State, 3 ardışık → sonraki güçlenir
2. **Glacial Spike** — 6m buz hat + %40 slow + Frost State+2, Fire üstünde → dondur + patlat
3. **Living Bomb** — hedefe bomba, ölünce 3 komşuya sek
4. **Blink** — 6m ışınlanma, düşman içinden geç → stun
5. **Frozen Orb** — yavaş buz küre + chill
6. **Prism Beam** — cursor kanal lazer, Light State'e göre güç
7. **Meteor** — 0.5s windup + büyük AoE knockdown
8. **Frost Wall** — buz bariyer + slow + radiant hasar
9. **Solar Flare** — radyant koni delici
10. **Blizzard** — 1s cast bölgesel slow

### Passive/Echo (S44) (2)
11. **Radiant Pillar** — yakın aura, Fire/Frost destekler
12. **Element Charge** — Fire spells 2x bedel + instant cast modifier
