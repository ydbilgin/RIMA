## Rule
Any non-Claude agent that edits tracked project files must commit its own work.

## Prefixes
- Codex: `CODEX: <what changed and why>`
- Antigravity: `ANTIGRAVITY: <what changed and why>`
- Other agents: `<AGENT_NAME>: <what changed and why>`

Claude edits directly via Claude Code and does not need manual attribution.
